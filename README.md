1. Factorial Program (Functions)

Purpose: 
This program calculates the factorial of a number using a user-defined function.

How it works:
- A function named "factorial(num)" is defined.
- The function uses a for loop to multiply numbers from 1 to the given number.
- The final factorial value is returned by the function.
- The user enters a number as input.
- The program prints the factorial of the entered number.
